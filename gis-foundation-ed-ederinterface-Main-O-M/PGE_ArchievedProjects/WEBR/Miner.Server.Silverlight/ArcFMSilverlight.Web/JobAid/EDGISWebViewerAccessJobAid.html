EDGIS Web Viewer Access

To get access to EDGIS Web Viewer:

•Create a Work Order using link http://wwwt2/MyITServices/forms/intake/2

•Fill out the below web form for access request

oProject: ED AM GIS

oProduct: ED GIS Web Viewer

oRequest Type: Access Request

oIssue: <Estimator/Mapper/Other>

•When the Access request has been fully processed by the Electric GIS Team, you will receive access to EDGIS Web Viewer and will be notified the access request has been completed successfully.

Where to Access EDGIS Web Viewer

•http://wwwedgis/EDViewer/
